# Style

Styles are built with `Vtiful\Kernel\Format` and applied to cells, rows or columns. A single `Format` handle can be reused across many cells.

Pages in this section:

* [Combination style](combination.md) — chain multiple style methods on one handle.
* [Global default style](default-format.md) — apply a style to every cell of a worksheet.

For the full catalogue of individual style attributes, see [Style list](../style-list/README.md).
